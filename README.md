流体云扩展 (Fluid Cloud Extension)
 
一款专为 ColorOS 系统打造的 Xposed 模块，旨在解锁系统 UI 的隐藏定制选项，让您的流体云、快速设置磁贴和状态栏更具个性。
 
⚠️ 本项目为个人维护的开源项目，完全免费，为爱发电。
如果您喜欢，欢迎 Star 或贡献代码！
 

 
✨ 功能亮点
 
- 流体云位置微调
自由调整流体云（挖孔周围区域）的左右边距，完美适配不同屏幕尺寸或用户习惯。
- 音乐封面形状切换
在圆形与圆角矩形之间切换音乐播放器的封面显示样式。
- 快速设置磁贴可用性控制
强制显示或隐藏静音磁贴与 OTG 磁贴，不受系统版本限制。
- 2x1 磁贴背景填满
让 2x1 大小的磁贴背景填满整个格子（透明效果），视觉更统一。
- 快速设置面板颜色微调（实验性）
修改部分颜色资源，例如让 2x1 小磁贴背景变为透明（需手动配置）。
- 一键重启 SystemUI
修改配置后无需重启手机，点击按钮即可立即生效（需 Root 权限）。
- 可视化配置界面
告别手动编辑文件，所有选项均可在 App 内通过开关和输入框调整。
 

 
📱 适用环境
 
- 系统要求：ColorOS（Android 15 及以上）
由于模块依赖系统特定类名，可能无法在所有版本上正常工作。
- 需要 Root 权限：用于重启 SystemUI 及写入系统级配置
- 需要 LSPosed / Xposed 框架：推荐使用 LSPosed 激活模块
- 存储权限：因配置文件暂存于共享目录，需授予所有文件访问权限
后续版本计划迁移至应用私有目录，降低权限要求。
 

 🚀 安装与使用
 
1.在 LSPosed 中启用模块，作用域勾选：
 com.android.systemui
 com.oplus.systemui.plugins
2.重启 SystemUI 或手机。
3.打开「流体云扩展」，授予所有文件访问权限。
4.在配置界面调整各项参数：
 流体云左右边距
 音乐封面形状
 静音/OTG 磁贴显示开关
 2x1 磁贴背景透明填满开关
5.点击 重启作用域 使配置生效。
 
💡 提示：
 
- 修改边距后建议点击 提交 保存

- 开关会自动保存到内存，但必须重启作用域才会写入并生效
 

 
🧩 配置项说明
 
- 左间距 | 输入框 left | 流体云左侧边缘距离屏幕左边缘的距离（像素）

- 右间距 | 输入框 right | 流体云右侧边缘距离屏幕右边缘的距离（像素）

- 音乐背景形状 | shape_switch | 开=圆角矩形，关=圆形

- 静音磁贴 | mute_switch | 开=强制显示，关=强制隐藏

- OTG 磁贴 | otg_switch | 开=强制显示，关=强制隐藏

- 2x1 磁贴背景填满 | bg2x1_switch | 开=填满且透明，关=系统默认
